# ESS457

This repository stores Jupyter notebooks for example calculations associated with the course ESS 457 / 567 Environmental Geochemisty. These notebooks should appear in your working directory of JupyterLab access from the course Canvas site. Run the notebooks to walk through the example problems. If you would like to modify the notebooks, it is reccomended to make a copy of the notebook rather than modifying the example notebook.
